## Getting Started

`install nvm or node version 18`

`nvm install 18`
_can skip if using without nvm_

`nvm use 18`
_can skip if using without nvm_

`npm install`
Install all the dependencies

### Run in Dev Mode

`npm run dev`

#### To Run Project

`npm run build`

`npm start`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
